# PreprocessingUCLA  
We provide a preprocessing pipeline for the UCLA data (https://openneuro.org/datasets/ds000030/versions/00016).

## Install ANTs
```bash
cd ~  
git clone https://github.com/stnava/ANTs.git  
mkdir ~/ANTs/bin  
cp ~/PreprocessingUCLA/code/build.sh ~/ANTs/bin 
cd ~/ANTs/bin  
sh build.sh
```
